# EasyJapanese

## 项目介绍

### 任务背景

近年来，高考及大学日语学习群体持续扩容，然而与之适配的学习资源却相对匮乏。在此背景下，我拟开发一款专注日语单词学习的
App。其功能丰富多样，不仅可便捷查询背诵日语单词，并依据记忆曲线精准设定复习周期，内置详尽的日语词典，提供单词精准释义与标准发音；还搭建了活跃的日语社区，集发帖、聊天、即时通讯、互相关注与私信互动于一体。同时，配备大量日语阅读材料，借助专门的文章发布系统助力阅读训练，以及日语电台强化听力练习，设有五十音图学习板块，涵盖各类日语考级资料与资讯、教材内容。此外，还设有用户背诵单词排行榜，能精准统计用户每日学习时长及单词量等数据，为日语学习者打造全方位、个性化、高效的学习平台。

### 成果形式

App；H5网页；微信小程序
